# Cookie 配置指南

## 问题说明

目前 yt-dlp 解析各平台视频时需要 Cookie 认证：
- ✅ 抖音：需要 Cookie
- ✅ 小红书：可能需要 Cookie
- ✅ B 站：需要 Cookie

## 获取 Cookie 的方法

### 方法 1：使用浏览器扩展（推荐）

#### Chrome/Edge 浏览器
1. 安装扩展：**Get cookies.txt LOCALLY** 或 **EditThisCookie**
   - Chrome 商店搜索：`cookies.txt`
2. 访问对应网站并登录
   - 抖音：https://www.douyin.com
   - B 站：https://www.bilibili.com
   - 小红书：https://www.xiaohongshu.com
3. 点击扩展图标，导出 `cookies.txt` 文件
4. 将文件重命名并放到服务目录：
   ```
   douyin_cookies.txt       # 抖音 Cookie
   bilibili_cookies.txt     # B 站 Cookie
   xiaohongshu_cookies.txt  # 小红书 Cookie
   ```
   存放位置：`/root/video-downloader/`

### 方法 2：手动复制 Cookie

1. 浏览器打开对应网站并登录
2. 按 F12 打开开发者工具
3. 找到 Network 选项卡，刷新页面
4. 点击任意请求，找到 `Cookie` 头
5. 复制完整的 Cookie 字符串

### 方法 3：使用 yt-dlp 提取

如果你已经登录了浏览器：
```bash
# 提取指定网站的 Cookie
yt-dlp --cookies-from-browser chrome --cookies cookies.txt "https://www.douyin.com"
```

## 配置 Cookie 文件

### 文件名规范
| 平台 | Cookie 文件名 |
|------|-------------|
| 抖音 | `douyin_cookies.txt` |
| B 站 | `bilibili_cookies.txt` |
| 小红书 | `xiaohongshu_cookies.txt` |

### 文件格式
Cookies 应该是 `cookies.txt` 格式（Netscape HTTP Cookie File）：
```
# Netscape HTTP Cookie File
# This file was generated by a browser extension
www.douyin.com	TRUE	/	TRUE	1234567890	__ac_nonce	abc123...
www.douyin.com	TRUE	/	TRUE	1234567890	sessionid	xyz789...
```

### 存放位置
```
/root/video-downloader/
├── douyin_cookies.txt       # 创建此文件
├── bilibili_cookies.txt     # 创建此文件
└── xiaohongshu_cookies.txt  # 创建此文件
```

## 注意事项

1. **Cookie 有效期**：Cookie 会过期，需要定期更新
2. **登录状态**：确保浏览器已登录对应账号
3. **隐私安全**：Cookie 包含登录信息，不要泄露
4. **多账号**：如需切换账号，重新导出 Cookie 覆盖文件即可

## 快速获取 Cookie（以 B 站为例）

```bash
# 1. 在浏览器登录 B 站
# 2. 使用扩展导出 Cookie
# 3. 或者使用以下命令（如果已安装 yt-dlp 和浏览器）
yt-dlp --cookies-from-browser chrome --cookies /root/video-downloader/bilibili_cookies.txt "https://www.bilibili.com"
```

## 无 Cookie 替代方案

如果无法获取 Cookie，可以考虑：
1. 使用第三方 API（如各类去水印 API）
2. 使用官方 API（需要注册开发者账号）
3. 使用其他解析服务
